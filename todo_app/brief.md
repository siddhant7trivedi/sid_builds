Hi! I need a simple To-Do List App built inside Claude.

## Core Requirements

- Users should be able to add a task by typing it in
- Each task should have a checkbox to mark it as done
- Users should be able to delete a task they no longer need

## How it should look

- Clean and simple — nothing fancy
- Tasks should visually look "crossed out" or greyed out when completed
- A basic input box with an "Add" button at the top

## How it works with end users

- Someone opens it and immediately understands what to do — no instructions needed
- It should feel responsive, like clicking a checkbox instantly marks it done
- Should work smoothly without any page reloads or glitches

## Explicit Constraints

> That's it for now — keep it simple! No categories, no due dates, no priorities. Just add, check off, and delete.
