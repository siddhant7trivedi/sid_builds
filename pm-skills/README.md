# PM Skills

A reference library of product management frameworks, workflows, and templates — built for Claude Code skills.

Each folder contains two files:
- `SKILL.md` — the framework, principles, and step-by-step workflow
- `TEMPLATE.md` — a ready-to-fill template for the deliverable
