## 7. Non-Functional Requirements
> Performance, security, scalability, accessibility, and reliability constraints.

- **Performance:** e.g. Response time < 2 seconds for 95% of requests
- **Security:** e.g. All data encrypted in transit and at rest
- **Scalability:** e.g. Must support up to 10,000 concurrent users
- **Accessibility:** e.g. WCAG 2.1 AA compliant
- **Availability:** e.g. 99.9% uptime SLA
