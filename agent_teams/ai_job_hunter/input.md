# Job Hunter — Search Input

Fill in your search criteria below, then share this with the Job Hunter Director.

---

## Search Criteria

| Field | Your Input |
|-------|-----------|
| **Job Title** *(Required)* | |
| **Companies** *(Optional — max 3, comma-separated)* | |
| **Cities** *(Required — max 3, comma-separated)* | |
| **Years of Experience** *(Optional)* | |
| **Posted Within** *(Pick one)* | `10 days` / `20 days` / `30 days` / `Above 30 days` |
| **Top Results** *(Optional — default: 5)* | |

---

## Example

| Field | Value |
|-------|-------|
| **Job Title** | Senior Software Engineer |
| **Companies** | Google, Meta, Apple |
| **Cities** | San Francisco, New York, Seattle |
| **Years of Experience** | 5+ years |
| **Posted Within** | 20 days |
| **Top Results** | 5 |

---

## Notes

- If **Companies** are provided, the agents will prioritize their official career pages.
- If **Companies** are left blank, the agents perform an open-market search across all sources.
- **Posted Within** controls how recent the listings must be. Use `Above 30 days` for a wider search.
- **Top Results** controls how many ranked jobs appear in `output/job_hunter.md`.
