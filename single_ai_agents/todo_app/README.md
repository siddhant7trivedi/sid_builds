# To-Do List App

A simple, focused task manager — no backend, no frameworks, no bloat.
Built to solve one problem: capture and clear tasks without friction.

## What It Does
- Add tasks via a text input
- Mark tasks complete with a checkbox (strikethrough + greyed out)
- Delete tasks with a single click
- No page reloads — everything happens in-browser instantly

## What It Doesn't Do (By Design)
No categories, no due dates, no priorities, no accounts.
Scope was intentionally constrained to nail the core loop first.

## Product Decisions
- **Font:** Inter — neutral, legible, nothing competes with the task list
- **Palette:** Off-white background, near-black text, muted slate for completed tasks
- **Signature detail:** Thin left-border accent on each task that disappears on completion — subtle visual cue that the task has lost its weight
- **No delete animation** — instant removal feels more decisive than a fade

## Stack
HTML · CSS · JavaScript · React
